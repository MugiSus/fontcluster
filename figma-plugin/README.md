# Fontcluster Apply Font

This Figma plugin applies the latest Fontcluster FontItem selection to the
selected Figma text nodes. If no text node is selected, it creates a text node
and applies the font there.

## Development

1. Start Fontcluster.
2. In Figma Desktop, import `manifest.json` from this directory:
   `Plugins > Development > Import plugin from manifest...`
3. Run `Fontcluster Apply Font` from Figma.
4. Click a FontItem in Fontcluster.

The plugin polls the local Fontcluster bridge at `http://localhost:38653/latest`
while it is open.

The plugin source is `code.ts`. Figma runs `dist/code.js`, generated by
`pnpm run figma:build` before importing or reloading the development plugin.
